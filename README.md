# Wiekanwat

Internal consultancy web application for tracking employee assignments and information.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Information Board: Overview of all employees with filtering
- Employee Profiles: Detailed view of individual employees
- Filtering: By role, skills, and current client

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Mock data (no backend yet)
